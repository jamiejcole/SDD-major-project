Welcome to the help page.

Here you can find information and FAQs on how to use all of the pages on this website.



What is the bar at the top?

That bar is the navigation bar. You can use it to navigate between all the pages on the website.

How do I use view your certificates for the GROK courses?

If you click on the Certificates page, you are able to view each section of Grok course and click the download button for the corresponding topic.

What is your JavaScript game?

By clicking on the JS GAME page in the navigation bar, you are able to play my version of the popular video game Flappy Bird. It is a simple game, where the objective is to jump through the random holes in the moving bar, and avoid crashing!

What are your mini projects?

If you click on the Projects page in the navigation bar, you are able to see my four mini projects and a description of each one. These projects were originally written in Python but have been transferred over to JavaScript and given a pretty GUI with HTML and CSS.

How do I see some of your code for your Grok courses?

All of the Grok courses on the Certificates page contain some sample code from my favourite project from the corresponding Grok course! You can see these here.
